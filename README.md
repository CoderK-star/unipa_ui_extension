# UNIPA UI ヘルパー

UNIVERSAL PASSPORT のブラウザ画面を、端末内だけの処理で使いやすくする Chrome / Edge 向け Manifest V3 拡張機能です。

## 主な機能

- 入力フォームの自動保存と、下書きの復元・破棄UI。
- `chrome.alarms` を使った、任意有効化のセッション維持。
- Shadow DOM で注入する軽量なパンくず・戻るナビゲーション。
- 掲示板の検索、締切順ソート、未読フィルタ、ローカル既読管理。
- PDFリンクのページ内プレビューと別タブ表示。
- 提出期限を検出して表示する締切ダッシュボード。
- 任意有効化の締切デスクトップ通知。
- 機能のON/OFF、確認間隔、通知時間、ローカルデータ削除を行うポップアップ設定。

ユーザーデータはすべて `chrome.storage.local` に保存されます。外部解析、クラウド同期、第三者サーバーへの送信は行いません。

## ローカルでの読み込み

1. `chrome://extensions` または `edge://extensions` を開きます。
2. デベロッパーモードを有効にします。
3. 「パッケージ化されていない拡張機能を読み込む」を選びます。
4. このプロジェクトフォルダー `C:\unipa_ui_extension` を選択します。

`file://` のサンプルHTMLで確認する場合は、拡張機能の詳細画面で「ファイルの URL へのアクセスを許可する」を有効にしてください。

## プロジェクト構成

- `manifest.json`: Manifest V3 の拡張機能マニフェスト。
- `src/background/service_worker.js`: セッション維持、締切通知、ローカルデータ削除。
- `src/content/`: ページ注入、自動保存、ナビゲーション、掲示板ツール、PDFプレビュー、締切ダッシュボード。
- `src/shared/`: 定数、ストレージ補助、汎用セレクタ。
- `src/popup/`: 設定ポップアップ。
- `icons/`: 通知用アイコン。

## 補足

- セッション維持は初期状態では無効です。ユーザーが明示的に有効化した場合のみ動作します。
- 締切デスクトップ通知も初期状態では無効です。ユーザーが明示的に有効化した場合のみ通知します。
- 現在は汎用的なUNIPA判定とDOMセレクタで動作します。実際のサンプルHTML確認後に、大学別・ページ別のDOMルールを追加してください。
