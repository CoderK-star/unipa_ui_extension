# UNIPA UI ヘルパー

UNIVERSAL PASSPORT のブラウザー体験をローカル環境だけで改善する、Chrome / Edge Manifest V3 拡張機能の Phase 1 MVP です。

## 主な機能

- 編集可能な入力欄のローカル自動保存と、復元・破棄UI
- `chrome.alarms` を使った任意有効化のセッション維持
- Shadow DOM で注入する軽量なパンくず / 戻るナビゲーション
- 自動保存、セッション維持、間隔、ローカルデータ削除を設定できるポップアップ

ユーザーデータはすべて `chrome.storage.local` に保存されます。外部解析、クラウド同期、第三者サーバーへの通信は行いません。

## ローカルでの読み込み方法

1. `chrome://extensions` または `edge://extensions` を開きます。
2. デベロッパーモードを有効にします。
3. 「パッケージ化されていない拡張機能を読み込む」を選びます。
4. このプロジェクトフォルダー `C:\unipa_ui_extension` を選択します。

`file://` のサンプル HTML で確認する場合は、拡張機能の詳細画面で「ファイルの URL へのアクセスを許可する」を有効にしてください。

## プロジェクト構成

- `manifest.json`: MV3 拡張機能マニフェスト
- `src/background/service_worker.js`: セッション維持アラームとローカルデータ削除
- `src/content/`: ページ注入、自動保存、ナビゲーション
- `src/shared/`: 定数、ストレージ補助、汎用セレクター
- `src/popup/`: 設定ポップアップ

## 補足

- セッション維持は既定で無効であり、ユーザーが明示的に有効化する必要があります。
- 初期実装では汎用的な UNIPA 判定とフォームセレクターを使用しています。大学固有の DOM ルールは、サンプル HTML を確認後に追加してください。
